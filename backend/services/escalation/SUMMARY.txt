================================================================================
                         SYSTEM ARCHITECTURE
================================================================================

ESCALATION FLOW:

  Incident Detected
         │
         ├─ Check 4 Factors:
         │  ├─ Local Capacity (>85% utilized?)
         │  ├─ Mass Casualty (≥3 patients OR CATASTROPHIC?)
         │  ├─ Response Time (>15 minutes?)
         │  └─ Dispatch Queue (>2x available units?)
         │
         ├─ Evaluate Factors:
         │  ├─ 0 Factors → LOCAL (no action)
         │  ├─ 1 Factor → DISTRICT (optional)
         │  ├─ 2 Factors → DISTRICT (recommended)
         │  └─ 3+ Factors → REGIONAL (required)
         │
         ├─ For District+ → Mutual Aid
         │  └─ Identify neighbors → Send requests → Collect responses
         │
         ├─ For MCI → Mass Casualty Handler
         │  └─ Declare → Triage → Register Patients → Transport
         │
         └─ Resource Pooling
            └─ Check pool → Request units → Allocate → Auto-return

INTEGRATION POINTS:

  EscalationRules ──→ Triggers MCI Handler & Mutual Aid Coordinator
                 └──→ Recommends resource pool requests
  
  MutualAidCoordinator ──→ Coordinates with neighboring districts
                      └──→ Manages unit arrival and return
  
  MassCasualtyHandler ──→ Manages patients and sectors
                     └──→ Requests resources for deployment
  
  ResourcePoolingManager ──→ Allocates cross-jurisdictional units
                        └──→ Auto-rebalances based on load

                          QUICK START
================================================================================

1. IMPORT AND INITIALIZE

   const { EscalationSystem } = require('./index');
   const system = new EscalationSystem();

2. MONITOR ESCALATION

   const escalation = system.getEscalationRules()
     .determineEscalationLevel(incidentData);
   
   if (escalation.recommended) {
     // Take action
   }

3. HANDLE MCI

   const mciId = system.getMCIHandler()
     .declareMCI(incidentData);
   
   system.getMCIHandler()
     .registerPatient(mciId, patientData);

4. REQUEST RESOURCES

   // Mutual Aid
   const maidRequest = await system.getMutualAidCoordinator()
     .requestMutualAid(requestData);
   
   // Or Resource Pool
   const poolAllocation = await system.getPoolingManager()
     .requestPooledResources(jurisdiction, requestData);

5. LISTEN FOR EVENTS

   system.getEscalationRules().on('mutual-aid-recommended', (data) => {
     console.log('Escalation triggered:', data);
   });

6. GET STATUS

   const status = system.getSystemStatus();
   console.log('Active MCIs:', status.activeMCIs);
   console.log('Mutual Aid Requests:', status.activeMutualAidRequests);

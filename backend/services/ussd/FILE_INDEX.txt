3. location-resolver.js (12.53 KB) - LOCATION DETECTION
   
   Resolves location from multiple sources
   
   Location Methods (priority order):
     1. Cell Tower Triangulation (±500m)
        • Uses MCC/MNC from network code
        • No GPS needed (ideal for rural)
     2. GPS Coordinates (±10m)
        • Direct latitude/longitude
     3. Network Operator Data (±50km)
        • Fallback to carrier region
     4. Reverse Geocoding
        • OpenCage API (Africa coverage)
        • Google Maps (fallback)
   
   Methods:
     • resolveLocation() - Main entry point
     • resolveCellTowerLocation() - Cell triangulation
     • resolveGPSLocation() - GPS resolution
     • reverseGeocode() - Coordinates to address
     • getLocationQuality() - Quality assessment
     • cacheLocation() - Cache with TTL
   
   Usage:
     const resolver = new LocationResolver(config);
     const location = await resolver.resolveLocation(
       phoneNumber, 
       networkCode
     );
     const quality = resolver.getLocationQuality(location);

4. sms-fallback.js (11.69 KB) - SMS FALLBACK DISPATCH
   
   SMS-based dispatch when USSD unavailable
   
   Features:
     • Intent Detection - Extract emergency type from SMS
     • Condition Extraction - Identify medical conditions
     • Confidence Scoring - Rate detection confidence
     • Auto-dispatch - Dispatch if confidence ≥ 0.8
     • Confirmation - Request YES/NO if 0.5-0.8
     • Clarification - Ask for details if < 0.5
   
   Methods:
     • handleIncomingSMS() - Process SMS message
     • analyzeMessageIntent() - Extract intent type
     • extractMedicalConditions() - Parse conditions
     • processSMSIntent() - Decision logic
     • sendConfirmationRequest() - Request confirmation
     • handleConfirmationResponse() - Process YES/NO
     • sendSMS() - Send via gateway
   
   Intent Types:
     • medical - Medical emergency
     • accident - Accident/collision
     • ambulance - General ambulance request
     • unknown - Unclear
   
   Medical Conditions Detected:
     • chest pain, breathing difficulty, unconscious
     • severe bleeding, burns, poison, fractures, etc.
   
   Usage:
     const sms = new SMSFallback(config);
     const intent = sms.analyzeMessageIntent('chest pain');
     const result = await sms.handleIncomingSMS(
       phoneNumber, 
       messageText
     );
